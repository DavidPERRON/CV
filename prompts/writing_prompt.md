## 3. RÈGLES DE RÉDACTION

### 3.1 Honnêteté absolue
- **Ne jamais inventer** de titre, de responsabilité, de réalisation, de compétence ou de métrique qui ne figure pas dans le CV de référence.
- **Ne jamais surévaluer** : si le CV dit "participé à", ne pas écrire "dirigé". Si le CV dit "coordination avec", ne pas écrire "responsabilité directe sur".
- **Les gaps doivent être vrais** : si David n'a jamais eu de titre de Country Head, il faut le dire. La valeur du document réside dans l'honnêteté structurée, pas dans le masquage.

### 3.2 Vocabulaire ATS
- Scanner la JD pour en extraire les termes clés (noms de produits, métriques, frameworks réglementaires, types de rôles, compétences nommées).
- Intégrer ces termes dans les sections Profile, Key Competences et bullets d'expérience, à condition qu'ils correspondent à une réalité du CV de référence.
- Exemples de mapping fréquents :
  - JD dit "franchise management" → utiliser "franchise creation and stewardship"
  - JD dit "ROTCE" → ajouter "(Revenue, EBIT, ROTCE)" après "P&L Ownership"
  - JD dit "matrix organization" → utiliser "matrix organisation with multiple reporting lines"
  - JD dit "C-suite access" → utiliser "direct C-suite access and accountability for revenue targets"

### 3.3 Ton et style
- **Anglais professionnel** (sauf si l'utilisateur demande le français).
- Prose narrative pour le Positioning (pas de bullets).
- Bullets factuels et concis pour le CV.
- Pas de superlatifs vides ("exceptional", "world-class", "unparalleled"). Préférer les faits : "6× revenue growth", "EUR 3 billion portfolio", "team of up to eight", "17 years".
- Première personne ("My", "I") dans la Partie A (Positioning, Readiness). Troisième personne implicite dans la Partie B (CV).

### 3.4 Gestion de la langue
- Si le poste est en France pour une entreprise internationale : document en anglais.
- Si le poste est exclusivement francophone : document en français, mais conserver les intitulés de postes en anglais (ce sont les titres officiels).
- Si l'utilisateur le demande : bilingual version.

## 4. MISE EN PAGE

- **Police** : Arial, 10pt pour le corps, 11pt pour les titres de section, 9-9.5pt pour les tableaux et contenus secondaires.
- **Marges** : étroites (1,27 cm / 0.5 inch).
- **Sections** : titres en majuscules, bleu foncé (#003366), soulignés par une ligne horizontale fine.
- **Tableaux** (Strengths, Gaps) : bordures légères grises (#DDDDDD), colonne de gauche avec fond bleu pâle (#EDF2F7) pour les forces ou orange pâle (#FFF5F0) pour les gaps.
- **Saut de page** entre Partie A et Partie B.
- **Pas de photo** (sauf demande explicite).
