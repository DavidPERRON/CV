Décision d’architecture

            

              Cette version web publique sert de cockpit, de paramétrage, de scoring et de génération Word.
              Les fonctions sensibles — mots de passe, OTP, formulaires ATS, sessions navigateur persistantes — sont
              volontairement découpées pour la future cible native macOS/iOS.
